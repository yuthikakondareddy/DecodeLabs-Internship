# Project 2: Credit Card Fraud Detection using Machine Learning

This project involves:
- Loading and exploring a credit card transaction dataset
- Splitting the dataset into training and testing sets
- Training a Logistic Regression model
- Predicting fraudulent transactions
- Evaluating the model using Accuracy Score, Confusion Matrix, and Classification Report

## Technologies Used:
- Python
- Pandas
- Scikit-learn

## Files included:
- project2.py (Python code)
- README.md (Project documentation)

## Dataset
The dataset used for this project is the **Credit Card Fraud Detection** dataset from Kaggle.

Download `creditcard.csv` from Kaggle and place it in the project folder before running the code.
