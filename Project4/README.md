# Project 4: NLP Sentiment Analysis

This project performs sentiment analysis on customer reviews using Natural Language Processing (NLP) and Machine Learning.

## Features
- Text preprocessing
- Stopword removal
- Lemmatization
- TF-IDF Vectorization
- Naive Bayes Classification
- Sentiment Prediction
- Model Evaluation using Accuracy, Confusion Matrix, and Classification Report

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn

## Files Included
- project4.py (Python source code)
- training.1600000.processed.noemoticon.csv (Dataset)
- README.md

## Dataset
The dataset used in this project is the **Sentiment140** dataset. If the dataset is too large to upload to GitHub, it can be downloaded from Kaggle:
https://www.kaggle.com/datasets/kazanova/sentiment140
